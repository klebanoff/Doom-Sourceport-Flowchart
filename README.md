# doom-sourceport-flowchart

To install dependencies:

```bash
bun install
```

To run (with hot reload and Bun bundler for TypeScript frontend):

```bash
bun run start
```

To type-check the project (strict TypeScript for `index.ts` and `src/`):

```bash
bun run typecheck
```

This project was created using `bun init` in bun v1.2.19. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

## Data source

Data about Doom source ports and related information is taken from [DoomWiki.org](https://doomwiki.org).
